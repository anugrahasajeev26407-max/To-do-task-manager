# ✦ Tasky — Modern To-Do Task Manager

A sleek, dark-themed task manager built with vanilla HTML, CSS & JavaScript.

## Features
- ✅ Add, edit, delete tasks
- 🏷️ Priority levels (High / Medium / Low)
- 📂 Category filters (Work, Personal, Health, Learning)
- 🔍 Live search & sort
- 📅 Due dates with overdue detection
- 📊 Progress tracker
- 💾 Data saved in localStorage (persists on refresh)
- 📱 Fully responsive

## How to Run Locally

1. Download or clone this repository
2. Open `index.html` in any browser — **no server needed!**

```bash
# Option: open directly
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

## Tech Stack
- HTML5, CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Google Fonts: Syne + DM Sans
- localStorage for persistence
